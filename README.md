# OpenClaw Sub-agents

A plug-and-play plugin for [OpenClaw](https://openclaw.io) that adds a **Sub-agents** tab to the Control UI. Monitor every spawned sub-agent in real time and spawn new AI agents from 14 categories with one click.

## Install

```bash
openclaw plugins install github:Soulz-io/OpenClaw---Sub-agents
openclaw gateway restart
```

Done. Open the Control UI and the **Sub-agents** tab appears automatically.

## Features

### Live Dashboard
- Real-time overview of all spawned sub-agents
- Status cards: total, running, completed, errors
- Click any agent to see full details (prompt, model, duration, session keys)
- Auto-refreshes every 5 seconds

### Spawn Agent
Click **+ Spawn Agent** to create a new AI agent in three steps:

1. **Category** &mdash; 14 domains (Design, Engineering, Marketing, Data, Legal, ...)
2. **Specialism** &mdash; choose a focus area (React, UX Design, SEO, Machine Learning, ...)
3. **Style** &mdash; pick a personality style (Startup, Enterprise, Indie Craft, ...)
4. Optionally describe a task, then hit **Spawn**

Agents are generated by the [Soulz](https://soulz.io) platform with unique names, personalities, and specialisations.

## Configuration

The plugin works out of the box for monitoring. To enable the **Spawn** feature, add your Soulz credentials in the Control UI config editor (Plugins > openclaw-subagents) or in `openclaw.json`:

```json
{
  "plugins": {
    "entries": {
      "openclaw-subagents": {
        "enabled": true,
        "config": {
          "soulzApiUrl": "https://api.soulz.io",
          "soulzApiKey": "your-api-key"
        }
      }
    }
  }
}
```

## API

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/plugins/openclaw-subagents/api/subagents` | GET | List all sub-agent records |
| `/plugins/openclaw-subagents/api/summary` | GET | Stats: total / running / done / error |
| `/plugins/openclaw-subagents/api/spawn` | POST | Spawn a new agent via Soulz |

### Spawn request body

```json
{
  "category": "software_engineering",
  "specialism": "React",
  "style_theme": "startup",
  "task_description": "Build a landing page component"
}
```

## How it works

The plugin uses zero-config injection &mdash; no file copying, no config changes required:

1. **HTTP handler** intercepts the Control UI HTML on the fly and injects a `<script>` tag
2. **Client-side injector** uses Shadow DOM + MutationObserver to add the tab to the navigation
3. **Dashboard** runs inside an iframe at `/plugins/openclaw-subagents/`
4. **Lifecycle hooks** track `subagent_spawned` and `subagent_ended` events in real time
